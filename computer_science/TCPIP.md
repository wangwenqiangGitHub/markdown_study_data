# TCP/IP协议族

> Transmission Control Protocol 与 Internet Protocol
>
> 可以翻译为传输控协议及因特网协议

TCP/IP协议就是为使用互联网而开发制定的协议族。因此，==互联网的协议就是TCP/IP，TCP/IP就是互联网的协议==。
